# 🤖 AskSumit AI

A production-ready Full Stack AI Assistant built with **Next.js, TypeScript, MongoDB, JWT Authentication, and Groq LLM**.

AskSumit AI is a personal AI agent capable of answering questions about my professional profile while protecting private information through role-based access control.

# ⚙️ Installation

```bash
git clone https://github.com/sumitraj7766/ask-sumit-ai.git

cd ask-sumit-ai

npm install

npm run dev
```

---

# Environment Variables

Create a `.env.local` file:

```env
MONGODB_URI=

JWT_SECRET=

GROQ_API_KEY=

EMAIL_USER=

EMAIL_PASS=
```

---

# 🚀 Build

```bash
npm run build
```
